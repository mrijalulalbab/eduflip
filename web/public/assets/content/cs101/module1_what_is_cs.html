Hardware vs Software

Hardware (Perangkat Keras)

Komponen fisik komputer yang bisa Anda sentuh:

  • CPU (Processor) - "Otak" komputer yang memproses instruksi
  • RAM - Memori sementara untuk menjalankan program
  • Storage (HDD/SSD) - Penyimpanan permanen untuk data
  • GPU - Chip khusus untuk grafis dan komputasi paralel
  • Motherboard - Papan utama yang menghubungkan semua komponen

Software (Perangkat Lunak)

Program dan instruksi yang berjalan di atas hardware:

  • Operating System (OS) - Windows, macOS, Linux, Android
  • Applications - Browser, Office, Games, dll.
  • Drivers - Software penghubung hardware dan OS

Bagaimana Komputer "Berpikir"?

Sistem Biner (Binary)

Komputer hanya memahami dua angka: 0 dan 1. Ini disebut sistem biner. Mengapa? Karena secara fisik, komponen elektronik hanya memiliki dua status: mati (0) atau hidup (1). 

Contoh Konversi Desimal ke Biner:

Desimal   Biner
-------   --------
0         0
1         1
2         10
3         11
4         100
5         101
10        1010
255       11111111

Bit dan Byte

  • 1 Bit = satu digit biner (0 atau 1)
  • 1 Byte = 8 bits = bisa menyimpan angka 0-255
  • 1 Kilobyte (KB) = 1024 Bytes
  • 1 Megabyte (MB) = 1024 KB
  • 1 Gigabyte (GB) = 1024 MB
  • 1 Terabyte (TB) = 1024 GB
üí° Contoh Nyata: Satu huruf (character) biasanya membutuhkan 1 byte. Jadi kata "Hello" membutuhkan 5 bytes untuk disimpan!

Tingkatan Bahasa Pemrograman

1. Low-Level Languages

Bahasa yang dekat dengan hardware, sulit dibaca manusia tapi sangat cepat:

  • Machine Code - Biner murni (0 dan 1)
  • Assembly - Mnemonic untuk instruksi CPU (MOV, ADD, JMP)

2. High-Level Languages

Bahasa yang mudah dibaca manusia, perlu dikompilasi atau diinterpretasikan:

  • Python - Mudah dipelajari, serbaguna
  • JavaScript - Bahasa web, interaktif
  • Java - Enterprise, cross-platform
  • C/C++ - Performa tinggi, sistem operasi
  • PHP - Web backend, seperti aplikasi EduFlip ini!
// Contoh Code "Hello World" di berbagai bahasa:

// Python
print("Hello World")

// JavaScript
console.log("Hello World");

// Java
System.out.println("Hello World");

// PHP
echo "Hello World";
