Apa Itu Jaringan Komputer?

Jaringan komputer adalah sekumpulan perangkat yang terhubung satu sama lain untuk berbagi data dan sumber daya. Jaringan memungkinkan komputer Anda berkomunikasi dengan jutaan komputer lain di seluruh dunia!

Jenis-Jenis Jaringan

Jenis Cakupan Contoh
PAN (Personal Area Network) Beberapa meter Bluetooth headset ke HP
LAN (Local Area Network) Gedung/kampus WiFi kantor, lab komputer
MAN (Metropolitan Area Network) Kota Jaringan antar kampus
WAN (Wide Area Network) Negara/dunia Internet!

Komponen Jaringan

Perangkat Keras (Hardware)

  • Router: Mengarahkan data antar jaringan berbeda
  • Switch: Menghubungkan perangkat dalam satu jaringan
  • Modem: Mengubah sinyal digital ke analog (dan sebaliknya)
  • Access Point: Menyediakan koneksi WiFi
  • Network Interface Card (NIC): Kartu jaringan di komputer

Media Transmisi

  • Kabel Ethernet (UTP/STP): Kabel tembaga twisted pair
  • Fiber Optic: Kabel serat optik, sangat cepat
  • Wireless (WiFi): Gelombang radio
  • Satelit: Untuk area terpencil

Alamat IP dan DNS

IP Address

Setiap perangkat di jaringan memiliki alamat IP yang unik, seperti alamat rumah untuk surat. 

IPv4 (32-bit):
Format: xxx.xxx.xxx.xxx
Contoh: 192.168.1.1

IPv6 (128-bit):
Format: xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx
Contoh: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Bagaimana Internet Bekerja?

Model OSI (7 Layers)

OSI Model menjelaskan bagaimana data bergerak dari aplikasi ke kabel fisik:

Layer Nama Fungsi Contoh
7 Application Interface ke user HTTP, FTP, SMTP
6 Presentation Enkripsi, kompresi SSL/TLS, JPEG
5 Session Mengelola koneksi NetBIOS
4 Transport Reliable delivery TCP, UDP
3 Network Routing antar jaringan IP, ICMP
2 Data Link Transfer dalam network Ethernet, WiFi
1 Physical Sinyal fisik Kabel, radio

HTTP Methods

Method Fungsi Contoh
GET Mengambil data Membuka halaman web
POST Mengirim data Submit form login
PUT Update data Edit profil
DELETE Hapus data Hapus posting

HTTP Status Codes

  • 200 OK - Berhasil
  • 301 Moved Permanently - Redirect permanen
  • 404 Not Found - Halaman tidak ditemukan
  • 500 Internal Server Error - Error di server

Keamanan Jaringan

Ancaman Umum

  • Malware: Virus, worm, trojan
  • Phishing: Email/situs palsu untuk mencuri data
  • DDoS: Serangan membanjiri server
  • Man-in-the-Middle: Penyadapan komunikasi

Perlindungan

  • HTTPS: Enkripsi komunikasi web (lihat ğŸ”’ di browser)
  • Firewall: Menyaring traffic masuk/keluar
  • VPN: Mengenkripsi seluruh koneksi internet
  • Password kuat: Kombinasi huruf, angka, simbol

Kesimpulan Modul 5

  • Jaringan menghubungkan komputer untuk berbagi data.
  • IP address adalah alamat unik setiap perangkat.
  • DNS menerjemahkan nama domain ke IP address.
  • HTTP adalah protokol web, HTTPS versi amannya.
  • Keamanan jaringan sangat penting di era digital.
