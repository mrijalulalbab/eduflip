Apa Itu Program?

Program adalah serangkaian instruksi yang ditulis dalam bahasa pemrograman untuk memberi tahu komputer apa yang harus dilakukan. Jika komputer adalah "mesin", maka program adalah "resep" yang mengontrol mesin tersebut.

üí° Analogi: Bayangkan komputer adalah koki yang sangat patuh tapi tidak bisa berpikir sendiri. Anda harus memberikan instruksi yang sangat detail dan tepat!

Variabel: Tempat Menyimpan Data

Variabel adalah wadah untuk menyimpan nilai. Seperti kotak berlabel yang bisa Anda isi dengan berbagai hal. 

// Contoh variabel dalam JavaScript

let nama = "Budi";           // String (teks)
let umur = 20;               // Number (angka)
let sudahLulus = false;      // Boolean (benar/salah)
let nilaiUjian = 87.5;       // Number (desimal)

Aturan Penamaan Variabel

  • Harus diawali huruf, underscore (_), atau dollar sign ($)
  • Tidak boleh mengandung spasi
  • Case-sensitive (Nama ‚â† nama)
  • Hindari kata-kata reserved seperti if, for, while

Operator: Melakukan Operasi

Operator Aritmatika

let a = 10;
let b = 3;

a + b   // 13 (Penjumlahan)
a - b   // 7  (Pengurangan)
a * b   // 30 (Perkalian)
a / b   // 3.33 (Pembagian)
a % b   // 1  (Modulus/sisa bagi)
a ** b  // 1000 (Pangkat: 10¬≥)

Operator Perbandingan

5 == 5    // true (sama dengan)
5 != 3    // true (tidak sama)
5 > 3     // true (lebih besar)
5 < 3     // false (lebih kecil)
5 >= 5    // true (lebih besar atau sama)
5 <= 3    // false (lebih kecil atau sama)

Operator Logika

true && true   // true  (AND: keduanya harus true)
true || false  // true  (OR: salah satu true cukup)
!true          // false (NOT: kebalikan)

Control Flow: Mengontrol Alur Program

1. Conditional (If/Else)

Menjalankan kode berdasarkan kondisi tertentu:

let nilai = 85;

if (nilai >= 80) {
    console.log("Grade: A");
} else if (nilai >= 70) {
    console.log("Grade: B");
} else if (nilai >= 60) {
    console.log("Grade: C");
} else {
    console.log("Grade: D");
}

// Output: "Grade: A"

2. Loops (Perulangan)

Mengulangi kode beberapa kali:

For Loop

// Mencetak angka 1 sampai 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
// Output: 1, 2, 3, 4, 5

While Loop

let hitungan = 0;

while (hitungan < 3) {
    console.log("Iterasi: " + hitungan);
    hitungan++;
}
// Output: Iterasi: 0, Iterasi: 1, Iterasi: 2
üí° Kapan Pakai Apa?
  • For loop: Ketika tahu berapa kali akan mengulang
  • While loop: Ketika pengulangan bergantung pada kondisi

Functions: Kode yang Dapat Dipakai Ulang

Function adalah blok kode yang diberikan nama dan bisa dipanggil berulang kali. Ini membantu menghindari penulisan kode yang sama. 

// Mendefinisikan function
function sapaPengguna(nama) {
    return "Halo, " + nama + "!";
}

// Memanggil function
console.log(sapaPengguna("Rina"));   // Output: Halo, Rina!
console.log(sapaPengguna("Budi"));   // Output: Halo, Budi!

Function dengan Multiple Parameters

function hitungLuasPersegiPanjang(panjang, lebar) {
    return panjang * lebar;
}

let luas = hitungLuasPersegiPanjang(5, 3);
console.log(luas);  // Output: 15
