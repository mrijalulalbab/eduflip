Bit dan Byte

Apa Itu Bit?

Bit (Binary Digit) adalah unit terkecil data dalam komputer. Bit hanya memiliki dua nilai: 0 atau 1.

Apa Itu Byte?

Byte adalah kumpulan 8 bit. Dengan 8 bit, kita bisa merepresentasikan 2^8 = 256 nilai berbeda (0-255). 

Contoh:
1 byte = 8 bits
00000000 = 0   (desimal)
00000001 = 1
00000010 = 2
00000011 = 3
...
11111111 = 255

Representasi Angka

Bilangan Bulat (Integer)

Komputer menyimpan integer dalam format biner. Ukuran tergantung tipe:

  • 8-bit: 0 to 255 (unsigned) atau -128 to 127 (signed)
  • 16-bit: 0 to 65,535 (unsigned)
  • 32-bit: 0 to ~4.3 miliar (unsigned)
  • 64-bit: Sangat besar! (~18 quintillion)

Bilangan Desimal (Floating Point)

Angka desimal seperti 3.14 disimpan menggunakan standar IEEE 754:

  • Sign bit: Positif atau negatif
  • Exponent: Pangkat bilangan
  • Mantissa/Fraction: Digit signifikan

Representasi Audio

Suara adalah gelombang analog. Komputer mengkonversinya menjadi digital melalui proses sampling:

  • Sample Rate: Berapa kali per detik suara "diukur" (e.g., 44.1 kHz)
  • Bit Depth: Presisi pengukuran (e.g., 16-bit)
  • Channels: Mono (1) atau Stereo (2)
