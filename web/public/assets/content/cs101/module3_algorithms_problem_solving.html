Apa Itu Algoritma?

Algoritma adalah serangkaian langkah-langkah yang terstruktur dan logis untuk menyelesaikan suatu masalah. Algoritma harus jelas, terbatas, dan menghasilkan output yang diharapkan.

ğŸ’¡ Karakteristik Algoritma yang Baik:
  • Input - Menerima data masukan (bisa 0 atau lebih)
  • Output - Menghasilkan minimal satu output
  • Definiteness - Setiap langkah jelas dan tidak ambigu
  • Finiteness - Algoritma harus berakhir (tidak infinite)
  • Effectiveness - Setiap langkah dapat dilakukan

Pseudocode: Menulis Algoritma dengan Bahasa Manusia

Pseudocode adalah cara menulis algoritma menggunakan bahasa yang mirip pemrograman, tapi lebih mudah dibaca manusia. Tidak ada aturan sintaks yang ketat.

Contoh: Menentukan Bilangan Ganjil/Genap

ALGORITMA CekGanjilGenap
INPUT: bilangan (integer)
OUTPUT: "Ganjil" atau "Genap"

MULAI
    BACA bilangan
    JIKA bilangan MOD 2 == 0 MAKA
        CETAK "Genap"
    JIKA TIDAK
        CETAK "Ganjil"
    AKHIR JIKA
SELESAI

Flowchart: Visualisasi Algoritma

Flowchart adalah representasi grafis dari algoritma menggunakan simbol-simbol standar:

  • Oval - Start/End
  • Parallelogram - Input/Output
  • Rectangle - Proses/Instruksi
  • Diamond - Keputusan (Ya/Tidak)
  • Arrow - Alur program

Langkah-Langkah Pemecahan Masalah

  1. Pahami Masalah - Apa yang diminta? Apa input dan output?
  2. Rencanakan Solusi - Tulis langkah-langkah kasar
  3. Bagi Menjadi Sub-Masalah - Pecah masalah besar jadi kecil
  4. Tulis Algoritma - Pseudocode atau flowchart
  5. Implementasi - Tulis kode program
  6. Testing - Uji dengan berbagai input
  7. Debug - Perbaiki jika ada error

Contoh Algoritma: Mencari Nilai Maksimum

Masalah

Diberikan array berisi angka, temukan nilai terbesar.

Pseudocode

ALGORITMA CariMaksimum
INPUT: array[] dengan n elemen
OUTPUT: nilai maksimum

MULAI
    SET maks = array[0]    // Anggap elemen pertama adalah terbesar
    
    UNTUK i = 1 SAMPAI n-1 LAKUKAN
        JIKA array[i] > maks MAKA
            maks = array[i]
        AKHIR JIKA
    AKHIR UNTUK
    
    KEMBALIKAN maks
SELESAI

Implementasi JavaScript

function cariMaksimum(arr) {
    let maks = arr[0];
    
    for (let i = 1; i < arr.length; i++) {
        if (arr[i] > maks) {
            maks = arr[i];
        }
    }
    
    return maks;
}

// Uji coba
let angka = [3, 7, 2, 9, 4, 1];
console.log(cariMaksimum(angka));  // Output: 9

Contoh Algoritma: Menghitung Jumlah Array

Pseudocode

ALGORITMA HitungJumlah
INPUT: array[] dengan n elemen
OUTPUT: jumlah semua elemen

MULAI
    SET total = 0
    
    UNTUK SETIAP elemen DALAM array LAKUKAN
        total = total + elemen
    AKHIR UNTUK
    
    KEMBALIKAN total
SELESAI

Implementasi JavaScript

function hitungJumlah(arr) {
    let total = 0;
    
    for (let elemen of arr) {
        total += elemen;
    }
    
    return total;
}

// Uji coba
let nilai = [10, 20, 30, 40];
console.log(hitungJumlah(nilai));  // Output: 100

Pattern Umum dalam Algoritma

1. Linear Search (Pencarian Linear)

Mencari elemen dengan memeriksa satu per satu:

function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) {
            return i;  // Ditemukan di index i
        }
    }
    return -1;  // Tidak ditemukan
}

2. Counter Pattern (Penghitung)

Menghitung kemunculan sesuatu:

function hitungVokal(str) {
    let count = 0;
    let vokal = "aeiouAEIOU";
    
    for (let char of str) {
        if (vokal.includes(char)) {
            count++;
        }
    }
    
    return count;
}

console.log(hitungVokal("Hello World")); // Output: 3

3. Accumulator Pattern (Akumulator)

Mengumpulkan hasil secara bertahap:

function factorial(n) {
    let result = 1;
    
    for (let i = 2; i <= n; i++) {
        result *= i;
    }
    
    return result;
}
