Struktur Data Fundamental

Array dan Linked List adalah dua struktur data paling dasar yang wajib dikuasai. Keduanya digunakan untuk menyimpan koleksi elemen, tetapi dengan cara yang sangat berbeda.

1. Array

Array adalah struktur data yang menyimpan elemen-elemen secara berurutan di memori. Bayangkan deretan loker di sekolah - semua bersebelahan dan bernomor.

Visualisasi Array

Index:    [0]    [1]    [2]    [3]    [4]
        +------+------+------+------+------+
Data:   |  10  |  20  |  30  |  40  |  50  |
        +------+------+------+------+------+
Memory:  0x100  0x104  0x108  0x10C  0x110  (berurutan!)

Keunggulan Array

  • Akses Cepat: O(1) - Langsung lompat ke index manapun
  • Cache Friendly: Data bersebelahan = CPU cache efisien
  • Sederhana: Mudah dipahami dan digunakan

Kelemahan Array

  • Ukuran Tetap: Harus ditentukan di awal (di bahasa low-level)
  • Insert/Delete Lambat: O(n) - Harus geser elemen lain

2. Linked List

Linked List adalah struktur data yang menyimpan elemen dalam "node" yang saling terhubung via pointer. Bayangkan kereta api - setiap gerbong terhubung ke gerbong berikutnya.

Visualisasi Linked List

  v

   Node 1             Node 2             Node 3

Implementasi Linked List (JavaScript)

class Node {
    constructor(data) {
        this.data = data;
        this.next = null;
    }
}

class LinkedList {
    constructor() {
        this.head = null;
    }
    
    // Insert di awal - O(1)
    prepend(data) {
        const newNode = new Node(data);
        newNode.next = this.head;
        this.head = newNode;
    }
    
    // Insert di akhir - O(n)
    append(data) {
        const newNode = new Node(data);
        if (!this.head) {
            this.head = newNode;
            return;
        }
        let current = this.head;
        while (current.next) {
            current = current.next;
        }
        current.next = newNode;
    }
}

Kesimpulan Modul 2

  • Array menyimpan data berurutan di memori - akses O(1), insert/delete O(n)
  • Linked List menyimpan data dalam node terpisah yang saling terhubung
  • Pilih struktur data berdasarkan operasi yang paling sering dilakukan
