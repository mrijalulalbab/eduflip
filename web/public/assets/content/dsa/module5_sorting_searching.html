Mengapa Sorting & Searching Penting?

Sorting (pengurutan) dan Searching (pencarian) adalah operasi paling fundamental dalam pemrograman.

BAGIAN A: SEARCHING

1. Linear Search

Cara paling sederhana: cek satu per satu dari awal sampai ketemu.

function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) {
            return i;
        }
    }
    return -1;
}

Kompleksitas: O(n)

2. Binary Search

Cara cerdas: bagi dua terus sampai ketemu. Syarat: data harus sudah terurut!

Visualisasi Binary Search

Mencari 23 dalam array [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]

Step 1: mid=16 ‚Üí 23 > 16, cari di kanan
Step 2: mid=56 ‚Üí 23 < 56, cari di kiri
Step 3: mid=23 ‚Üí KETEMU!

function binarySearch(arr, target) {
    let left = 0;
    let right = arr.length - 1;
    
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        
        if (arr[mid] === target) {
            return mid;
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return -1;
}

Kompleksitas: O(log n) - Sangat cepat!

BAGIAN B: SORTING

1. Bubble Sort

Bandingkan elemen bersebelahan, tukar jika salah urutan.

function bubbleSort(arr) {
    const n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}

Kompleksitas: O(n¬≤)

2. Merge Sort ‚≠ê

Divide and Conquer: bagi array jadi dua, sort masing-masing, lalu gabungkan. 

function mergeSort(arr) {
    if (arr.length <= 1) return arr;
    
    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));
    
    return merge(left, right);
}

function merge(left, right) {
    const result = [];
    let i = 0, j = 0;
    
    while (i < left.length && j < right.length) {
        if (left[i] < right[j]) {
            result.push(left[i++]);
        } else {
            result.push(right[j++]);
        }
    }
    
    return [...result, ...left.slice(i), ...right.slice(j)];
}

Kompleksitas: O(n log n)

Kesimpulan Modul 5

  • Linear Search O(n), Binary Search O(log n) tapi butuh sorted data
  • Bubble/Selection/Insertion Sort = O(n¬≤) - untuk edukasi
  • Merge Sort dan Quick Sort = O(n log n) - untuk produksi
