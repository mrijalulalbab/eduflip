Struktur Data Abstract (ADT)

Stack dan Queue adalah Abstract Data Types - artinya, mereka mendefinisikan perilaku (apa yang bisa dilakukan), bukan implementasi (bagaimana caranya).

1. Stack (Tumpukan)

Stack bekerja dengan prinsip LIFO (Last In, First Out). Elemen terakhir yang masuk adalah yang pertama keluar.

üí° Analogi: Bayangkan tumpukan piring di dapur. Piring terakhir yang ditaruh di atas adalah yang pertama diambil.

Operasi Utama Stack

  • push(item) - Menambah elemen ke atas stack - O(1)
  • pop() - Menghapus dan mengembalikan elemen paling atas - O(1)
  • peek() - Melihat elemen paling atas tanpa menghapus - O(1)
  • isEmpty() - Mengecek apakah stack kosong - O(1)

Implementasi Stack (JavaScript)

class Stack {
    constructor() {
        this.items = [];
    }
    
    push(element) {
        this.items.push(element);
    }
    
    pop() {
        if (this.isEmpty()) return "Stack is empty";
        return this.items.pop();
    }
    
    peek() {
        return this.items[this.items.length - 1];
    }
    
    isEmpty() {
        return this.items.length === 0;
    }
}

2. Queue (Antrian)

Queue bekerja dengan prinsip FIFO (First In, First Out). Elemen pertama yang masuk adalah yang pertama keluar.

üí° Analogi: Bayangkan antrian di kasir supermarket. Orang pertama yang mengantri adalah yang pertama dilayani.

   FRONT                                    REAR

Implementasi Queue (JavaScript)

class Queue {
    constructor() {
        this.items = [];
    }
    
    enqueue(element) {
        this.items.push(element);
    }
    
    dequeue() {
        if (this.isEmpty()) return "Queue is empty";
        return this.items.shift();
    }
    
    front() {
        if (this.isEmpty()) return "Queue is empty";
        return this.items[0];
    }
    
    isEmpty() {
        return this.items.length === 0;
    }
}

Perbandingan Stack vs Queue

üîµ Stack (LIFO)

  • Last In, First Out
  • Operasi di satu ujung saja (top)
  • Cocok untuk: Undo, Recursion, Backtracking

üü¢ Queue (FIFO)

  • First In, First Out
  • Operasi di dua ujung (front & rear)
  • Cocok untuk: Scheduling, BFS, Buffering

Kesimpulan Modul 3

  • Stack = LIFO (Last In First Out) - operasi push dan pop
  • Queue = FIFO (First In First Out) - operasi enqueue dan dequeue
  • Keduanya memiliki operasi O(1) untuk insert/delete
  • Pilih berdasarkan urutan pemrosesan yang dibutuhkan
